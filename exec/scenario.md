# 4. 시연 시나리오 (화면/클릭 상세)

## 4-1. 시연 준비
- FE 접속: `https://i14c209.p.ssafy.io`
- BE API: `https://i14c209.p.ssafy.io/api`
- WebSocket: `https://i14c209.p.ssafy.io/api/ws`
- 준비 계정
  - 호스트(카카오 로그인 1개)
  - 게스트(카카오 로그인 1개 또는 게스트 입장)

## 4-2. 데스크톱 시연 (기본 플로우)
### 1) 메인 진입
- 화면: `Desktop Main`
- 동작: 좌측 상단 로고 확인, 메인 CTA 버튼(예: “방 만들기/입장”) 클릭

### 2) 카카오 로그인
- 화면: 로그인 모달
- 동작: “카카오 로그인” 버튼 클릭 → 카카오 인증 완료 → 콜백 화면 진입 → 메인 복귀

### 3) 방 생성 (AI 체크리스트 포함)
- 화면: `RoomModal > CreateRoomForm`
- 입력 항목(예시)
  - 방 이름, 쇼핑 목적, 인원수, 목표/최소 예산, 관심 카테고리, 특성(트레잇)
- 동작: “방 생성” 클릭 → LLM 응답 대기 → 방 생성 완료

### 4) 초대 링크 공유
- 화면: 방 내부 상단/우측 패널
- 동작: “초대 링크” 복사 버튼 클릭 → 링크 공유

### 5) 게스트 입장
- 화면: `RoomModal > JoinRoomForm` 또는 초대 링크 접근
- 동작: 링크 접속 → 닉네임 입력 → 입장

### 6) 실시간 화면/커서/스크롤 동기화 확인
- 화면: `VideoChat` 또는 방 메인
- 동작
  - 게스트 마우스 이동 → 호스트 화면에서 커서 표시 확인
  - 페이지 스크롤 → 서로 동기화 확인

### 7) 실시간 채팅
- 화면: 우측 `ChatPanel`
- 동작: 메시지 입력 → 전송 → 상대방 화면에 즉시 표시 확인

### 8) 상품 리스트 및 장바구니
- 화면: `ProductList` 또는 상품 카드 섹션
- 동작
  - 상품 카드에서 “담기” 클릭
  - 우측 장바구니 패널에서 품목/참여자 표시 확인

### 9) 투표 생성/참여
- 화면: `VotePanel`
- 동작
  - “투표 만들기” 클릭 → 옵션 입력 → 생성
  - 참여자들이 옵션 선택 → 실시간 집계 확인

### 10) 정산(영수증/분할)
- 화면: `Settlement`
- 동작
  - “영수증 업로드” 클릭 → 이미지 선택 → OCR 분석 결과 확인
  - 품목별 참여자 분배 조정 → “정산 완료” 클릭

### 11) 정산 결과 확인
- 화면: `SettlementResult`
- 동작: 사용자별 금액/분배 내역 확인

## 4-3. 모바일 시연 (요약)
### 1) 모바일 메인
- 화면: `Mobile Main`
- 동작: 초대 링크 접속 또는 “방 입장” 버튼 클릭

### 2) 모바일 화상/실시간 기능
- 화면: `Mobile Video Chat`
- 동작
  - 카메라/마이크 토글
  - 채팅/투표/장바구니 패널 스와이프 또는 하단 탭 전환

### 3) 모바일 영수증 OCR
- 화면: `Mobile Receipt OCR`
- 동작: 영수증 촬영/업로드 → OCR 결과 확인

## 4-4. 시연 포인트 체크리스트
- 실시간 기능: 커서/스크롤/채팅/투표가 즉시 반영되는지
- WebRTC: 참가자 영상/오디오 정상 수신
- OCR: 이미지 업로드 → 분석 결과 반환
- 정산: 품목별 분배 및 최종 결과 화면
